GOLD Treatment Groups

A
mMRC 0–1 or CAT <10 · 0 exacerbations/year
Initial: Bronchodilator (LAMA preferred)
Escalate: Switch to alt bronchodilator or add LABA
Eos: ICS not indicated
B
mMRC ≥2 or CAT ≥10 · 0 exacerbations/year
Initial: LAMA + LABA
Escalate: Consider triple if persistent symptoms
Eos: ICS if eos ≥300 and symptoms persist
E
Any symptoms · ≥1 moderate/severe exacerbation/year
Initial: LAMA + LABA (or triple if eos ≥300)
Escalate: Triple → roflumilast or azithromycin
Eos: ICS if eos ≥100; strongly indicated ≥300
